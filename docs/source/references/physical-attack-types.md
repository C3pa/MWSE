# Physical Attack Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.physicalAttackType.*` namespace. For example, `tes3.physicalAttackType.thrust` has a value of `3`.

ID | Physical Attack Type
-- | --------------------
0  | None
1  | Slash
2  | Chop
3  | Thrust
4  | Projectile
5  | Creature1
6  | Creature2
7  | Creature3
