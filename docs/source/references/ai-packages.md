# AI Packages

!!! tip
	These values are available in Lua by their identifier in the `tes3.aiPackage.*` namespace. For example, `tes3.aiPackage.follow` has a value of `3`.

ID | AI Package
-- | ---------------
-1 | None
0  | Wander
1  | Travel
2  | Escort
3  | Follow
4  | Activate
