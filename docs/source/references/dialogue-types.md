# Dialogue Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.dialogueType.*` namespace. For example, `tes3.dialogueType.greeting` has a value of `2`.

ID | Dialogue Type
-- | -----------------
0  | Topic
1  | Voice
2  | Greeting
3  | Service
4  | Journal
