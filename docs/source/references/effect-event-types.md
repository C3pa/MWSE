# Effect Event Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.effectEventType.*` namespace. For example, `tes3.effectEventType.modStatistic` has a value of `3`.

ID | Effect Range
-- | -----------------
0  | Bool
0  | Boolean
1  | Int
1  | Integer
2  | Float
3  | Mod Statistic
