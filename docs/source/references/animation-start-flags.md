# Animation Start Flags

!!! tip
	These values are available in Lua by their identifier in the `tes3.animationStartFlag.*` namespace. For example, `tes3.animationStartFlag.immediateLoop` has a value of `2`.

ID | Flag
-- | ---------------
0  | Normal
1  | Immediate
2  | Immediate Loop
