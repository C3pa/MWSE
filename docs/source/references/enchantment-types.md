# Enchantment Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.enchantmentType.*` namespace. For example, `tes3.enchantmentType.constant` has a value of `3`.

ID | Enchantment Type
-- | -----------------
0  | Cast Once
1  | On Strike
2  | On Use
3  | Constant
