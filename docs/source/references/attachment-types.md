# Attachment Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.attachmentType.*` namespace. For example, `tes3.attachmentType.activator` has a value of `9`.

ID | Attachment Type
-- | ---------------
3  | Lock
6  | Variable
8  | Actor
9  | Activator
