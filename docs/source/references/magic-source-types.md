# Magic Source Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.magicSourceType.*` namespace. For example, `tes3.magicSourceType.alchemy` has a value of `3`.

ID | Magic Source Type
-- | -----------------
1  | Spell
2  | Enchantment
3  | Alchemy
