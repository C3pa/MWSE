# Specialization Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.specialization.*` namespace. For example, `tes3.specialization.magic` has a value of `1`.

ID | Specialization Type
-- | ---------------------
-1 | Invalid
0  | Combat
1  | Magic
2  | Stealth
