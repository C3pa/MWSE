# Voiceovers

!!! tip
	These values are available in Lua by their identifier in the `tes3.voiceover.*` namespace. For example, `tes3.voiceover.intruder` has a value of `2`.

ID | Voiceover
-- | ---------------------
0  | Hello
1  | Idle
2  | Intruder
3  | Thief
4  | Hit
5  | Attack
6  | Flee
