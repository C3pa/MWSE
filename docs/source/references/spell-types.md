# Spell Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.spellType.*` namespace. For example, `tes3.spellType.curse` has a value of `4`.

ID | Spell Type
-- | ---------------------
0  | Spell
1  | Ability
2  | Blight
3  | Disease
4  | Curse
5  | Power
