# Sound Mix Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.soundMix.*` namespace. For example, `tes3.soundMix.footsteps` has a value of `3`.

ID | Sound Mix Type
-- | ---------------------
0  | Master
1  | Voice
2  | Effects
3  | Footsteps
4  | Music
