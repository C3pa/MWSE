# Dynamic Effect Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.dynamicEffectType.*` namespace. For example, `tes3.dynamicEffectType.textureEffect` has a value of `4`.

ID | Dynamic Effect
-- | -----------------
0  | Ambient Light
1  | Directional Light
2  | Point Light
3  | Spot Light
4  | Texture Effect
