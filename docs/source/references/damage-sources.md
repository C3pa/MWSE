# Damage Sources

!!! tip
	These values are available in Lua by their identifier in the `tes3.damageSource.*` namespace. For example, `tes3.damageSource.shield` has a value of `shield`.

ID           | Damage Source
------------ | ---------------
script       | Script
fall         | Fall
suffocation  | Suffocation
attack       | Attack
magic        | Magic
shield       | Shield
