# Weather Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.weatherType.*` namespace. For example, `tes3.weatherType.rain` has a value of `4`.

ID | Weather Type
-- | ---------------------
0  | Clear
1  | Cloudy
2  | Foggy
3  | Overcast
4  | Rain
5  | Thunder
6  | Ash
7  | Blight
8  | Snow
9  | Blizzard
