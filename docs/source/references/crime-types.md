# Crime Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.crimeType.*` namespace. For example, `tes3.crimeType.theft` has a value of `5`.

ID | Crime Type
-- | -----------------
1  | attack
2  | killing
3  | stealing
4  | pickpocket
5  | theft
6  | trespass
7  | werewolf
