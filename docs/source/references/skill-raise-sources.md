# Skill Raise Sources

!!! tip
	These values are available in Lua by their identifier in the `tes3.skillRaiseSource.*` namespace. For example, `tes3.skillRaiseSource.book` has a value of `book`.

ID        | Skill Raise Source
--------- | ------------------
book      | Book
progress  | Progress
training  | Training
