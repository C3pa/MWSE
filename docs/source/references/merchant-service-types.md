# Merchant Service Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.merchantService.*` namespace. For example, `tes3.merchantService.spells` has a value of `0x800`.

ID       | Merchant Service Type
-------- | ---------------------
0x800    | Spells
0x4000   | Training
0x8000   | Spellmaking
0x10000  | Enchanting
0x20000  | Repair
