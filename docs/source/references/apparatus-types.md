# Apparatus Types

!!! tip
	These values are available in Lua by their identifier in the `tes3.apparatusType.*` namespace. For example, `tes3.apparatusType.calcinator` has a value of `2`.

ID | Apparatus
-- | ---------------
0  | Mortar and Pestle
1  | Alembic
2  | Calcinator
3  | Retort
